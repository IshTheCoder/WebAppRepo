src.analysis.json_to_DF module¶

src.analysis.json_to_DF.json_to_DF(fname, FILTER_OUT=[])¶

Takes NBA json file and returns dataframes of it for better computation NOTE: need to include .json extension in the string; opens from current directory NOTE: FILTER_OUT expects a list of strings that filters out column data by specified header

Header names to use can be found via: some_json_dict[‘resultSets’][0][‘headers’] headers = [‘SEASON_ID’, ‘PLAYER_ID’, ‘PLAYER_NAME’, ‘TEAM_ID’, ‘TEAM_ABBREVIATION’, ‘TEAM_NAME’, ‘PLAY_TYPE’, ‘TYPE_GROUPING’, ‘PERCENTILE’, ‘GP’, ‘POSS_PCT’, ‘PPP’, ‘FG_PCT’, ‘FT_POSS_PCT’, ‘TOV_POSS_PCT’, ‘SF_POSS_PCT’, ‘PLUSONE_POSS_PCT’, ‘SCORE_POSS_PCT’, ‘EFG_PCT’, ‘POSS’, ‘PTS’, ‘FGM’, ‘FGA’, ‘FGMX’]

Param

fname, str

Param

DataFrame
